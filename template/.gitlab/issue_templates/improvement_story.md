# Improvement story

## _We have \<current situation>, we want to have \<desired situation>_

### What's wrong now?

- _What's the current situation?_
- _What are the undesired outcomes?_

### What do we want?

- _What's the desired situation?_
- _What are the desired outcomes?_

### Are there any dependencies or constraints?

- _Do we need outside help in order to finish this ticket successfully?_

### How are we going to verify our solution?

- _What are the acceptance criteria?_
- _Is the solution (unit) testable?_
- _Is there an existing metric that will be influenced by the solution or do we need to create a new one?_
