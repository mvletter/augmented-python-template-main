# Python Template

## How to use

Make sure you have [Copier](https://copier.readthedocs.io/en/latest/) installed, you need the latest which at time of writing is version `9.2.0`

To start a new project from scratch:

```bash
copier copy git@gitlab.wearespindle.com:holodeck/templates/python-template.git {NAME OF PROJECT FOLDER}
```

To update a project to the latest version of the template:

```bash
cd {NAME OF PROJECT FOLDER}
copier update --conflict inline
```

For more information have a look at [this notion page](https://www.notion.so/theoracle/Python-Template-40776b05763142d293357c867224f588)

Or follow the tutorial after [setting up a new service.](https://noto.li/HVj9ZT)
