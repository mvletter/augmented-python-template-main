# Pitfalls

Knowledge base of problems encountered and how to prevent them.

## Categories

- `data-` - Data loss, corruption, wrong values
- `async-` - Timing, order, waiting issues
- `security-` - Passwords, access, authentication
- `ui-` - Buttons, screens, user interface
- `api-` - Communication between systems
- `state-` - Remembering things, caching, sessions
- `error-` - Error messages, crashes, handling

## How to Add

Use `/retro [description]` to analyze a bug and add it here automatically.
